◆ ProcessPhysics()

override State UnicornWait.ProcessPhysics ( double delta)
inlinevirtual

Count down the time left in the wait state. If the time is smaller or equal to zero, call funciotn SelectNextAttack to determine which attack is to be performed next and then change to that attack. If the unicorn remains in the wait state because the time is not up yet, the unicorn moves towards the current position of the player slowly.
