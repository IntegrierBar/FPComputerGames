◆ OnAreaEntered()

void MeleeAttackHurtBox.OnAreaEntered ( Area2D area)

The slime should only be able to detect collisions with the PC. If the slime collides with the PC, the damage function is called. Afterwards Monitoring is set to false again to prevent the player from taking damage from the same attack a second time.
