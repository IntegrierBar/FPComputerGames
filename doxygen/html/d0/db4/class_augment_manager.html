Detailed Description

This pseudo singleton class manages all the different Augments and AugmentEffect. It loads all AugmentEffects at runtime and is used to automatically create a random augment with 1-3 AugmentEffects

It handles all complicated parts of Augment creation.

It is an autoload of the Game and thus always accessible using the static Instance member
