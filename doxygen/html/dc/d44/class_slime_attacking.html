override State SlimeAttacking.ProcessPhysics ( double delta)
inlinevirtual

Once the attack animation is done, return to the Moving state and set monitoring of the melee attack hurt box to false so that the slime does not damage the player if they are colliding. While in Attacking state, move the slime according to the velocity set in the attack (no movement for ranged, to the position of the player for melee).
