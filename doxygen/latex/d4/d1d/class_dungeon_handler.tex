\doxysubsection{Detailed Description}
Manages the dungeon layout, room transitions, and player movement between rooms. 

This class is responsible for handling the overall structure of the dungeon, including loading rooms, managing room transitions, and keeping track of the player\textquotesingle{}s position within the dungeon.
