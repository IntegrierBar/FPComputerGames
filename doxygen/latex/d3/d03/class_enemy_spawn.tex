\doxysubsection{Detailed Description}
The \doxylink{class_enemy_spawn}{Enemy\+Spawn} class represents a node in the room where enemies can be spawned. These nodes are spread around the room and initialized by the \doxylink{class_room_handler}{Room\+Handler}. They set the initial values for slimes when they are spawned.
