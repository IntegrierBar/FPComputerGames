Allows to set the properties of the slime. This needs be called whenever a new slime is added to a scene. Set magic type, slime size and slime attack range of the slime. Furthermore the position of the slime in the dungeon is set here. If the slime size is large, the collision shapes of the slime are scaled up. The default collision shapes fit the small slime. Also set view range and attack range for the slime.
