First, if there is a player, check if the slime is close enough to change state. If not, reduce time\+Left. If time\+Left reaches zero, the slime changes from staying idle in one place to randomly walking in one direction or the other way around. This is done by calling the Change\+Random\+Walk function. Afterwards, move\+\_\+and\+\_\+slide is called so that the slime can move.
