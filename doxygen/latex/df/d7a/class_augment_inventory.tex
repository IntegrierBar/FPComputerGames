\doxysubsection{Detailed Description}
The \doxylink{class_augment_inventory}{Augment\+Inventory} is the root node of the augment inventory. It handles the creation of all Inventory\+Slots and the adding of new augments to the inventory. The scene is a child of the player so that it is always loaded as the scene containes all augment data. Usually the visibility and the processing is disabled. Except if the player clicks the "{}\+Open Augment Inventory"{} button. It can then be closed again using ESC.
