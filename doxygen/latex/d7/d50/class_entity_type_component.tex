Enum for the different types of entities. 

The different types are None, \doxylink{class_player}{Player}, Enemy, Boss, Projectile, \doxylink{class_slime}{Slime}, and \doxylink{class_unicorn}{Unicorn}. This enum uses flags, meaning an entity can be a combination of types. For example, a \doxylink{class_slime}{Slime} is both a \doxylink{class_slime}{Slime} and an Enemy.

NOTE\+: This enum is currently not in use. This is a design question if we want to have every relevant node have an \doxylink{class_entity_type_component}{Entity\+Type\+Component}.
