\doxysubsection{Detailed Description}
A room exit node that emits a signal when the player enters the door. Once a detection with the player is detected, a signal is emitted with the direction from which the player entered the door, which is used by \doxylink{class_dungeon_handler}{Dungeon\+Handler} to load the next room.
