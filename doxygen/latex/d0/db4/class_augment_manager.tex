\doxysubsection{Detailed Description}
This pseudo singleton class manages all the different Augments and \doxylink{class_augment_effect}{Augment\+Effect}. It loads all Augment\+Effects at runtime and is used to automatically create a random augment with 1-\/3 Augment\+Effects

It handles all complicated parts of \doxylink{class_augment}{Augment} creation.

It is an autoload of the Game and thus always accessible using the static Instance member
